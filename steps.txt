Globally install uv using pip
`pip install uv`
Go to your project folder, then run these in the terminal.
First installed python 3.12 as it is the latest supported by mediapipe
`uv python install 3.12`
`uv venv --python 3.12`
`source .venv/bin/activate`
This create a .venv virtual environment with name as your project folder
Install requirements
uv pip install -r requirements.txt
